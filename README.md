# Dorcas Ventures — Home of Quality Plasters

Prototype website for Dorcas Ventures, a premium plastering and ceiling materials company based in Accra, Ghana.

## Deployment

### Option 1 — Netlify Drop (fastest, no account needed)
1. Go to [netlify.com/drop](https://netlify.com/drop)
2. Drag and drop `index.html`
3. Share the live URL with your client

### Option 2 — GitHub Pages
1. Create a new GitHub repository
2. Upload `index.html` and `README.md`
3. Go to **Settings → Pages → Source → main branch**
4. Your site will be live at `https://yourusername.github.io/repo-name`

### Option 3 — Vercel
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → Import Project
3. Select your repo — Vercel auto-deploys on every push
